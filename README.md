## Introduction

Kids Center is designed to offer a variety of activities and learning resources for children. It includes games, educational content, and interactive activities to keep kids engaged and learning.

## Features

- **Educational Games**: Fun and interactive games that help children learn various subjects.
- **Learning Resources**: A collection of educational materials and resources.
- **Interactive Activities**: Activities that encourage creativity and critical thinking.
- **Parental Controls**: Features that allow parents to monitor and control their child's usage.

## Installation

To install the Kids Center project, follow these steps:

1. Clone the repository:
  ```bash
  git clone https://github.com/ouhssini/kids-center.git
  ```
2. Navigate to the project directory:
  ```bash
  cd kids-center
  ```
3. Install the dependencies:
  ```bash
  npm install
  ```

## Usage

To start the project, run the following command:
```bash
npm run dev
```
This will start the development server and you can access the application at `http://localhost:5173`.
